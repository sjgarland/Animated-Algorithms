The way in which information is arranged affects the ways in which it can be accessed: it's more efficient to look for a name in a list that is ordered alphabetically than in one that is unordered. The process by which a list is arranged in a order is known as sorting.

Sorting algorithms are widely studied, partly because they have important applications and partly because there are many ways in which they can be constructed. Some of these algorithms are simple and easy to understand, but relatively inefficient. Others (described later) proceed by problem reduction and are decidedly more clever and more efficient.

Elementary algorithms for sorting lists

  • Bubble (or exchange) sort
  • Insertion sort
  • Selection sort

Advanced algorithms for sorting lists

  • Merge sort
  • Quicksort
  • Heapsort
